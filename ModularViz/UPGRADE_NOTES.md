# ModularViz 全面优化（2026-06）

本次在保持原有使用方式（直接 `file://` 打开 `ModularViz.html`、classic script + 全局命名空间）
和**完全兼容旧数据文件**的前提下，对功能、交互、视觉与代码结构做了一轮整体升级。

## 数据兼容性（重点保证）

- 旧版 `v1.0 / v1.1` 普通 JSON 文件 **原样可读**；加载器现在更宽容：
  - `edges` 缺失 → 视为空数组；
  - 坐标为字符串 → 自动转数字；缺失的 `color` / `content` / `tag` → 取默认值；
  - 重复 id、缺 id 的节点、指向不存在节点的悬空边 → 自动跳过（旧版会直接渲染异常）。
- 旧版 `v2.0` 加密文件（`.enc.json` + `.keys.json`）加解密逻辑**未改动**，原样可读。
- 新保存的文件为 `version: "1.2"`，仅新增可选 `view: {tx, ty, sc}` 字段（保存视口位置）；
  旧文件没有该字段时加载后自动「适配视图」（fit-to-content）。
- 已用 Node 冒烟测试验证：v1.1 读取、v1.2 round-trip、视口恢复、撤销/重做、自动保存恢复。

## Bug 修复 / 性能

- **边标签 KaTeX 每帧重渲染**：拖动节点时每条带标签的边每帧都重新跑 KaTeX → 现在缓存
  `_renderedTag`，仅文本变化时重渲染，拖动只更新位置。
- **`Ctrl+C` / `Ctrl+N` 误触发模式切换**：快捷键现在正确区分修饰键。
- “New Node” / “Connect” 按钮现在均为开关式（再次点击退出模式）。
- 图加载改为同步（原先边在 rAF 中延迟加载），消除了快照/自动保存可能读到半加载状态的竞态。
- 面板编辑器的 mirror div + ResizeObserver 在直接关闭面板时未清理 → 已修复。
- 加载文件后自动适配视图（旧文件）或恢复保存时的视口（新文件）。

## 代码结构

- **统一图加载管线**：原先 `IO._applyGraph`、`CryptoIO`、`LatexImport._loadToCanvas` 三处重复的
  加载/清空逻辑合并为 `IO.applyGraph()` / `IO.buildPayload()` 单一入口（含校验、视口、事件）。
- **事件总线扩展**：所有图变更点统一发 `graph:changed`，加载发 `graph:loaded`，视口变化发
  `view:changed`（rAF 节流）。History / Persist / ViewUI 均为纯订阅者，互不耦合。
- 新增模块：`History.js`（快照式撤销栈，微任务合并 + 去重）、`Persist.js`（自动保存）、
  `ViewUI.js`(缩放条/计数器/空提示)、`BoxSelect.js`（框选）。
- 启动顺序：`Persist.init()`（恢复会话）→ `History.init()`（以恢复后的状态为撤销基线）。

## 视觉 / 可访问性

- 右下角缩放控制条、左下角计数器、空画布引导插画（含暗色模式全套配色）。
- 框选矩形、连线虚线预览样式。
- 撤销/重做按钮禁用态；所有按钮增加 `:focus-visible` 键盘焦点环。
- 搜索下拉键盘高亮行样式；触屏下缩放条按钮放大到 40px 触达标准。
- 品牌名统一为 ModularViz。

## 第二轮：节点显示与 LaTeX 编辑体验（2026-06）

### 节点卡片

- 内容预览从「剥离 `$` 的单行纯文本」升级为**真正的 KaTeX 渲染预览**（多行、含公式）。
- 截断算法保证**永不切断公式**（`LX.renderPreview`，约 240 字符预算；过大的公式块整体保留或
  整体丢弃，已用独立单测验证 6 种边界情况）。
- 超出最大高度时底部渐隐（仅在真正溢出时加 `clipped` 蒙版，短内容不受影响）。
- 卡片视觉精修：分层阴影、悬浮微升起、标题/预览分隔线、最大宽度放宽到 280px。
- **渲染缓存**：title/content 未变化时跳过 KaTeX 重渲染；尺寸变化自动触发边重布线。
- **双击节点直接进入重命名**（聚焦面板标题输入框并全选）。

### LaTeX 编辑（侧栏 + 全屏编辑器）

- 新模块 `LatexEditor.js`：
  - **全屏分栏编辑器**（源码 | 实时预览），面板里点「⛶ Expand」打开；Esc / Done 关闭；
    打开期间屏蔽全局快捷键（Delete / Ctrl+Z 不会误伤画布）。
  - **代码片段工具栏**（侧栏与全屏共用 12 个按钮）：行内/块级公式、分式、根号、上下标、
    求和、积分、矩阵、aligned 对齐环境、加粗、斜体——包裹选中文本并保留原生撤销栈
    （`execCommand` + `setRangeText` 兜底）。
  - **Tab 键插入缩进**（不再跳焦点）。
- 侧栏编辑器：编辑时**光标所在段落自动高亮同步到预览**（防抖跟随，不再只靠按钮）；
  编辑器聚焦光环；行号沟槽配色优化；节点卡片刷新与预览渲染合并防抖（打字不卡）。
- 渲染质量：KaTeX 常用宏（`\R \N \Z \Q \C \F \eps \abs{} \norm{} \set{} \inner{}`，
  仅渲染期生效，文件内仍是标准 LaTeX）；公式错误标红（`#ef4444`）；
  预览区**长公式横向滚动**而非溢出裁剪；预览排版（行高/字号/留白）整体优化。

## 架构约束（沿用）

仍保持 classic script + 全局命名空间（非 ES modules）：项目需要直接以 `file://` 打开使用，
ES modules 在 file 协议下会被浏览器 CORS 拦截。KaTeX 仍走 CDN，离线时优雅降级为纯文本。
